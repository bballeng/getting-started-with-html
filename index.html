Important Concepts

About the web

A web browser allows a user to make request to servers using HTTP protocol. Servers respond to the requests with the selected HTML files that the browser displays. The web is a collection of HTML documents.

HTML

Hypertext Markup Language (HTML) is the basis of any webpage. The text is what is displayed on a webpage and the markup is what the content looks like or how it is arranged. HTML also has references to other documents such as images or videos.

HTML Markup

Elements are composed of opening and closing tags.
anchor (a) tags make links.
image (img) tags are used for displaying images. The alternate text is required. It is the text that is displayed if a url does not exist. Images do not have closing tags because it does not have content,this is called a void tag.

All whitespace (tabs, paragraphs, spaces) in HTML turn into a single space when displayed on a webpage.
